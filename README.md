## JavaScript Practice Projects 🚀
A collection of small, fun, and experimental JavaScript projects built to master vanilla JS concepts, DOM manipulation, and asynchronous programming.
## 📂 Repository Structure
Each folder contains a standalone project with its own HTML, CSS, and JavaScript files.

* 01-todo-app/ - Task management with local storage.
* 02-weather-dashboard/ - Fetching live data from third-party APIs.
* 03-calculator/ - Modern grid layout with complex math logic.
* 04-whack-a-mole/ - Timing events, CSS animations, and score tracking.

## 🛠️ Tech Stack

* HTML5 - Semantic structure.
* CSS3 - Custom properties, Flexbox, and Grid.
* JavaScript (ES6+) - Core logic, DOM events, and Fetch API.

## 🎯 Key Learning Objectives

* Mastering DOM selection and event listeners.
* Understanding this, closures, and scope.
* Working with asynchronous code using Promises and Async/Await.
* Managing state without external frameworks.

## 🚀 How to Run These Projects

   1. Clone this repository to your local machine:
   
   git clone https://github.com
   
   2. Open any project folder.
   3. Launch the index.html file directly in your browser, or use the VS Code Live Server extension.
